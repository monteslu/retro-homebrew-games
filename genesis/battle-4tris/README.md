# Battle 4Tris

Block-stacking action for Sega Genesis / Mega Drive.

## Features
- 1 Player mode vs increasing speed
- 2 Player competitive battle
- Ghost piece preview
- Hard drop (C button)
- Combo scoring system
- Screen shake on line clears

## Controls
- **D-Pad**: Move piece
- **A**: Rotate clockwise
- **B**: Rotate counter-clockwise
- **C**: Hard drop
- **START**: 1 Player / Pause
- **A (title)**: 2 Players

## Scoring
| Lines | Multiplier |
|-------|------------|
| 1 | 1x |
| 2 | 3x |
| 3 | 5x |
| 4 | 8x |

Combos add bonus points!
